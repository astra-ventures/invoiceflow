# InvoiceFlow - Vercel Deployment Guide

## One-Click Deploy for Josh

### Prerequisites
- Vercel account connected to the astra-ventures GitHub organization
- Push access to this repository

### Deployment Steps
1. Go to [vercel.com/new](https://vercel.com/new)
2. Import this repository: `astra-ventures/invoiceflow`
3. **No environment variables needed** - app runs entirely client-side
4. **No build settings changes needed** - Next.js detected automatically
5. Click **Deploy**

### Post-Deploy
- Custom domain: Point to `invoiceflow.app` (if owned)
- Vercel will auto-deploy on every push to `main` branch
